# سیستم رزرواسیون رستوران 
سیستمی جهت رزرو میز توسط کاربران و مدیران رزروها توسط مدیران رستوران.
دارای سه پنل زیر:
- پنل کاربران
- پنل مدیران رستوران
- پنل ادمین

## موارد مورد نیاز 
موارد زیر باید روی سیستم نصب و تنظیم شده باشند:
- Laravel Herd
- PHP 8.4
- Composer
- Node.js 22
- MySQL 8

**درصورتی که Laravel Herd نصب شده باشد نیازی به نصب PHP و Composer نیست.**

## نصب و راه اندازی 
در ابتدا از فایل `.env.example` کپی گرفته و تنظیمات لازم را در آن قرار دهید.  

جهت نصب و راه‌اندازی از دستورات زیر استفاده نمایید:

```bash
composer install
npm install
php artisan key:generate
php artisan migrate --seed
```

جهت ورود به بخش ادمین از آدرس `/admin` استفاده نمایید.
